Installation
============

Required dependencies
---------------------

- `numpy <http://www.numpy.org/>`__
- `pandas <http://pandas.pydata.org/>`__
- `joblib <http://joblib.readthedocs.io/>`__
- `scipy <https://www.scipy.org/>`__


Instructions
------------

Install with

.. code-block:: bash

    conda install --channel conda-forge pyspatialstats
